# Plan: Transformación de Von Bastik - Página Profesional de Artistas

## Contexto del Proyecto

**Sitio actual:** Von Bastik Tattoo - estudio de tatuajes y piercings con diseño dark/gold.
**Objetivo:** Transformar en una página profesional centrada en presentar a los artistas tatuadores (Selene, Bo, Aitor) con fichas individuales completas, manteniendo la paleta actual (negro #0a0a0a, dorado #c9a96e, gris claro #e5e5e5).

**Estructura de archivos existente:**
- `index.html` - página principal actual (monolítica con CSS inline)
- `public/images/` - assets globales (banner.jpg)
- `public/artist/` - directorio vacío para imágenes de artistas

## 7. Esquema de Colores y Tipografía

### Paleta (mantener actual):
| Color | Hex | Uso |
|-------|-----|-----|
| Negro fondo | `#0a0a0a` | Fondo principal |
| Dorado primario | `#c9a96e` | Acentos, CTAs, links, iconos |
| Gris claro texto | `#e5e5e5` | Texto principal |
| Gris medio | `#888888` | Texto secundario |
| Gris oscuro cards | `#111111` | Fondo de tarjetas |
| Borde sutil | `#1a1a1a` | Bordes y separadores |

**Razón:** El negro transmite elegancia y seriedad; el dorado evoca premium y artesanal; el gris claro garantiza legibilidad sin fatiga visual.

### Tipografías:
- **Títulos:** Space Grotesk (ya usada) — geométrica, moderna, transmite profesionalismo artístico
- **Cuerpo:** Inter (ya usada) — alta legibilidad, optimizada para pantalla, neutral y profesional

## 12. Notas Técnicas

- **CSS:** Extraer todo el inline actual a `css/style.css`, mantener las mismas clases y añadir nuevas para las secciones adicionales
- **JS:** Mínimo necesario — toggle menú móvil, scroll reveal, lightbox galería, filtros de galería, tabs (si se mantienen)
- **HTML semántico:** Usar `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`
- **Accesibilidad:** ARIA labels, roles, focus visible, skip navigation link
- **Performance:** CSS en `<head>`, JS al final del `<body>` o con `defer`, imágenes con `loading="lazy"` y `decoding="async"`
